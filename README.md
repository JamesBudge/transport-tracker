# Transport Tracker — GitHub Pages package

This folder is ready to publish to GitHub Pages.

## Publish to GitHub Pages
1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Save.
6. Wait for the Pages URL to appear.

## Add to iPhone Home Screen
1. Open the GitHub Pages URL in **Safari** on your iPhone.
2. Tap **Share**.
3. Tap **Add to Home Screen**.
4. Confirm the name.

## Backup recommendation
Use **Export backup (.json)** regularly.
That file can later be re-imported with **Import backup (.json)**.
